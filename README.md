Sistema de gestão de estoque

O objetivo desse projeto é ser um sistema que cadastra, gerencia e audita estoques de produtos de uma loja.

As rotas principais serão:
    • Login
    • Cadastro de usuário
    • Cadastro de produto
    • Manutenção de estoque
    • Orçamento

Após a confecção do MVP, serão trabalhados:
    • Relatório de vendas
    • Configurações
    • Cadastro de multiplos estoques
    • Cadastro de multiplos preços
    • Promoção 

Esse projeto utilizará o framework Angular como motor do Frontend, assim como Express com Knex para Backend.
